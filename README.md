# SadeBiL Question Automation

This repository refreshes SadeBiL remote question banks from curated offline data and Wikidata CC0 facts.

- Daily scheduled workflow builds Turkish and English banks.
- Wikidata facts are preferred, then the curated bank fills every category/difficulty to 1000 questions.
- Quality checks must pass before publishing.
- Successful runs publish to Cloudflare Pages project `sadebil-questions`.
